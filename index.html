About

Multimedia Artist & Creative Technologist — Syntetika Project

Perjalanan saya berangkat dari ketertarikan pada audiovisual melalui animasi, VJ performance, dan video mapping. Seiring waktu, praktik saya berkembang dari penciptaan visual menuju eksplorasi hubungan antara visual, suara, teknologi, dan ruang. Saya tertarik pada bagaimana media tidak hanya menyampaikan informasi, tetapi juga mampu membentuk pengalaman, membangun atmosfer, dan menciptakan interaksi baru antara manusia dengan lingkungannya.

Melalui Syntetika Project, saya mengeksplorasi teknologi sebagai medium kreatif dengan menggabungkan visual interaktif, synthesizer, real-time media, spatial experience, dan instalasi multimedia. Praktik saya berfokus pada pengembangan sistem dan pengalaman yang responsif terhadap ruang serta audiens, melalui berbagai pendekatan seperti audiovisual performance, video mapping, projection art, interactive installation, generative visual system, dan creative technology development.

Bagi saya, teknologi bukanlah tujuan akhir, melainkan instrumen untuk memahami bagaimana manusia merasakan, bergerak, dan terhubung melalui sebuah pengalaman bersama. Melalui setiap eksperimen dan kolaborasi, saya terus mengembangkan pendekatan yang berada di persimpangan seni, teknologi, dan ruang, dengan tujuan menciptakan pengalaman multimedia yang adaptif, imersif, dan bermakna.

Project Diary

Documenting experiments, breakthroughs & implementation notes

09 JUL 2026

Website & Documentation Hub Launched

Launched syntetika.github.io as the central hub for all Syntetika Studio projects. Built with pure HTML/CSS/JS, hosted on GitHub Pages. This site now serves as both a project diary and a distribution hub for experimental tools.

#meta#documentation#web
03 JUL 2026

Syntetika Studio v1.1.1 — HRTF Overload Hotfix & Project Cleanup

Released v1.1.1 with critical fixes: threading safety in SpatialRender, refactor of spatial audio system (legacy removal), save/load improvements, UI fixes, and cleanup of orphan files. Memory leak and buffer overflow fixes across multiple modules (Push2Control, SpatialRender, GIFAnimator).

#syntetika-studio#release#bugfix#spatial-audio
15 JUN 2026

Syntetika Engine v1.1.0 — Audio Lag Fix & Performance Optimization

Major update for the browser-based DAW: resolved audio lag issues, optimized Web Audio API rendering pipeline, improved MIDI latency, and enhanced AI composition engine. Now running smoothly for live performance with hardware MIDI controllers.

#syntetika-engine#release#webaudio#midi
01 JUN 2026

HLTN Advanced Output — Initial OBS Plugin Development

Started development on the OBS Studio plugin for multi-display projection mapping. Implemented core features: warp mesh with interactive vertex editing, edge blending for seamless multi-projector setups, and auto-layout grid system. Built with C++ and CMake for Windows.

#hltn-mapping#obs-plugin#projection-mapping#c++
