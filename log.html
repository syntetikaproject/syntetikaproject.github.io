v1.1.1 Syntetika Studio 09 JUL 2026

HRTF Overload Hotfix & Project Cleanup

Problems Fixed

  • Threading safety issue di SpatialRender — race condition saat multiple nodes mengakses HRTF processor secara bersamaan
  • Memory leak di Push2Control — object tidak di-release saat disconnect
  • Buffer overflow di SpatialRender dan GIFAnimator — out-of-bounds write saat buffer size berubah runtime

Changes

  • Refactor sistem spatial audio — menghapus legacy HRTF path yang tidak terpakai
  • Perbaikan sistem save/load — state preservation antar sesi
  • UI improvements dan cleanup orphan files
v1.1.0 Syntetika Engine 15 JUN 2026

Audio Lag Fix & Performance Optimization

Problems Fixed

  • Audio lag saat playback lebih dari 2 menit — buffer scheduling tidak sinkron dengan AudioContext.currentTime
  • MIDI latency tidak konsisten — timing issue di Web MIDI API handler

Changes

  • Optimasi Web Audio API rendering pipeline — mengurangi callback overhead
  • Enhanced AI composition engine — rule-based pattern generation lebih responsif
  • Stabil untuk live performance dengan hardware MIDI controllers
